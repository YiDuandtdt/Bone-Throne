# BoneThrone / 骸骨王座

Unity 6.3 LTS 2.5D low-poly DND-style turn-based tactics demo.

## Modes
- Singleplayer: one local player controls four heroes.
- LAN Multiplayer: exactly four players, host-authoritative.

## Tech baseline
- Unity 6.3 LTS
- URP / Universal 3D
- uGUI + TextMeshPro
- Netcode for GameObjects + Unity Transport for LAN phase

## Development rule
This project uses phase-based Codex development. See /Docs and AGENTS.md.
